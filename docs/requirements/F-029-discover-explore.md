# F-029 Discover / Explore

The user discovers content beyond their pen pal network.

## Flow

When user opens U-066 DiscoverScreen:
- S-005 FeedService returns featured and trending posts.
- U-067 ExploreGrid displays the posts.

When user taps a post in U-067 ExploreGrid:
- App opens that post for viewing.

When user follows a poster from the post view:
- See F-024 Add Pen Pal.
