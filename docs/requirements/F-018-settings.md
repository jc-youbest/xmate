# F-018 Settings

The user adjusts app-wide preferences.

## Flow

When user opens U-025 SettingsScreen:
- The screen displays sections: pen defaults, paper defaults, appearance (dark mode), finger-as-pen toggle, account (see F-022), privacy (see F-038), about.

When user changes any preference on U-025 SettingsScreen:
- C-001 NoteStore (settings) persists the value.
- The app applies the change immediately where applicable (e.g., dark mode, finger-as-pen).
